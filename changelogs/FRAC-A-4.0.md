## A4.0

# Changes
- Loads of mod tweaks removals and additions
- Update Yuki Textures
- Added missing Placers from Immersive Petrolium and Immersive Engineering to Placer: Immersive Engineering Mod
- Beds can no longer be used to skip night 
- Wandering villager now disable
- Phantoms now disabled 
- Changed Coke Brick recipe
- Removed Engineers Blocks Blueprint, constant crashing
- Froststeel blocks now a cold block for Thermo Generators
- 5x Thermo Generators power production
- Increase LV and MV wire capacity 
- Squuezer, Metal Press and Workbench now use 75% of the power than before
- Changed Forgotten Upgrade Smithing Template dupe recipe
- Added Void fuel
- Added Void Stone
- Added Void Planks
- Added Void Chest
- Added Void Block
- Added Void Ingot
- Changed void questline chapter a lot
- Void Portal now uses Void Stone and Void Fuel to light 
- Void Sandstone recipe no longer drops in world
- Portal travel time reduced to 20 ticks
- Removed Strainers Ore Pieces recipes 
- Added Upgrade Base recipe
- Added Overworld Portal Frame and Overworld Catalyst
- Gated Flight Blocks, Resource Gens and Cloches
- Skill costs changed, iron colors 1 point, gold 3 points, diamond 5 points and emerald 10 points
- After respawning from a death you will now gain a debuff
- Removed X ray packs 
- Coins now have a fractured texture, i hate it 
- Updated Information Questline

# Fixes
- Fixed Reservoirs not spawning correctly 
- Eroding and Purifying water buckets now show in JEI
- Obsidian can no longer be used to make a nether portal
